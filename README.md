# ArcadeTracker

ArcadeTracker is a web application for tracking arcade achievements, badges, milestones, and leaderboard progress. Built with Next.js, it provides a modern dashboard for users to monitor their progress and access resources.

## Features
- Badge tracking and display
- Milestone progress visualization
- Leaderboard and stats dashboard
- Swag progress tracking
- FAQ and resource pages
- Calculator for arcade points

## Tech Stack
- **Frontend:** Next.js, React, TypeScript, CSS
- **Backend:** API routes (Next.js)
- **Styling:** PostCSS

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rushikesh-bobade/demo-project.git
   cd demo-project
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the App
```bash
npm run dev
# or
yarn dev
```
The app will be available at `http://localhost:3000`.

### Building for Production
```bash
npm run build
```
